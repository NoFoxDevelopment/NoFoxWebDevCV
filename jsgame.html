Features of this Game

In under 200 lines of JavaScript and 2 hours of coding/debugging I have accomplished the following with this small project:

  • A short, playable Pong-like game using JS and HTML5 Canvas Drawing (2d)
  • Score Keeping (a la simplified Tennis) and Win/Loss conditions
  • Mouse Tracking for Player Racket control
  • Simple "Shaky Paddle" AI script
  • AI which improves as the player scores and slows down as it scores.
  • Gradual increase in ball speed with each hit, promoting quick resolution of set
Source code for this game can be found HERE.

Planned updates/features include:
  • Random Start Trajectory for ball via Math.Rand x and y velocities
  • AI experimentation for smoother AI behavior. (Current ideas include decreasing speed with proximity to ball so AI never truly achieves "true center". This could lead to more challnging gameplay as AI would theoretically be more likely to hit at more extreme angles.)
  • More satisfying ball bounce. (currently, ball hitting racket edges can lead to "WTF" moments wherein the player believes they should have returned the ball.)
